# Techdegree-project-7: React Gallery App

 In this project:
 * Use `Create React App` tool to set up initial project
 * Use `JavaScript` and `JSX` to build out the gallery components in a modular fashion.
 * Use `React Router` to set up routes for three default topic pages and a search page.
 * Use the Fetch API to fetch data from the `Flickr API` and use it to display images in the app.
 * Add logic to handle the search and various requirements of the project.
 * Add to the supplied `CSS` to personalize the project.

## Getting Started

To get up and running with this project:
 * Run `npm install` to install all project's dependencies
 * Run `npm start` to run the app
 * Open http://localhost:3000 to view it in the browser.
